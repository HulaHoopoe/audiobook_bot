# Telegram-бот для преобразования текстовых файлов в аудиокниги

## Описание
Бот принимает текстовые файлы книг и превращает их в аудиокниги с естественной интонацией.  
Поддерживается выбор языка и типа голоса.

## Основной функционал
- **Приём файлов**: `TXT`, `EPUB`, `PDF` (до 50 МБ)  
- **Извлечение текста** из загруженных файлов  
- **Озвучка с использованием ИИ-голосов** на базе [Edge TTS](https://learn.microsoft.com/azure/cognitive-services/speech-service/text-to-speech)  
- **Выбор голоса**: русский/английский, мужской/женский  
- **Отправка готовых аудиофайлов** пользователю прямо в Telegram

## Технические требования
- **Python**: версия 3.12
